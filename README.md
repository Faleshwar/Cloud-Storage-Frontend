# CloudStorage â€“ Cloud Storage Frontend

CloudStorage is a modern cloud storage frontend built using **React** and **Tailwind CSS**.  
It provides a clean, responsive, and user-friendly interface for managing files and folders, similar to Google Drive or Dropbox.

This frontend communicates with a **Spring Boot backend** via REST APIs.

### Search & Navigation

### UI / UX
- Responsive design (mobile, tablet, desktop)
- Clean dashboard layout
- Tailwind CSS styling
- Toast notifications for actions
